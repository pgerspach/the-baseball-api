Player IDs

To query by player ID, make the request to your desired/{endpoint} and add the player's unique playerID to the end. Every player's playerID in The Baseball API is formed using the following formula:

[first 5 letters of the player's last name] + [first 2 letters of the player's first name]+[uniqueness number, with a leading 0 if single-digit]

For example, if you want to get statistics for Derek Jeter, you would need his player ID which is formed as "jeter"+"de"+"01" = "jeterde01"

Please note that for some players, such as Bryce Harper, there are players that precede them with the same pattern, which causes the final two digits of their playerID to be higher than 01. Bryce Harper is "harpbr03"

Endpoints

As of now, The Baseball API only serves 3 main endpoints:

  • /players
  • /pitching
  • /batting


/players

Make your request to /players/{player id} to retrieve basic personal information

Request:


            $.get("xxxxx/player/jeterde01").then(res=>{
                console.log(res)
            })
